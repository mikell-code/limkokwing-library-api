# Limkokwing Library API

A FastAPI-based Library Management System for Limkokwing University.

## Features
- View all books
- Search books by title, author, or category
- Borrow books
- Return books
- Check overdue books and fines

## Installation

pip install -r requirements.txt

## Run Project

uvicorn main:app --reload

## Open API Docs

http://127.0.0.1:8000/docs
